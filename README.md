# Flutter Widgets Catalog

（WIP）



### 计划

1、使用`Flutter`开发一个全平台的`Flutter Widgets Catalog APP`，并且开源。在这个APP中可以通过图形化的方式查看所有`Widgets`的介绍，使用说明，视频教程。

2、所有文档内容由前一天从`flutter.dev`官网爬取，保持和官网同步。

3、每一个`Widget`都录制一个视频教程。
